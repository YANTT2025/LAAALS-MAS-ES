# LAAALS-MAS-ES
This repository provides the implementation of LAAALS-MAS-ES, a meshless method for computing cutoff wavenumbers of two-dimensional PEC polygonal waveguides using locally adaptive auxiliary sources.

## Code and Data Availability

The MATLAB implementation of LAAALS-MAS-ES and the example files are available at the following Baidu Netdisk link:
